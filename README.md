# Movie Search App

A movie search application built with **React and TypeScript** that allows users to search for movies using an external API.

The project was created to practise working with API requests, managing loading states, implementing pagination, and building React applications with TypeScript.

---

## Features

- Search movies by title
- Fetch movie data from an external API
- Loading state while fetching results
- Pagination to browse additional results
- Responsive user interface

---

## Tech Stack

- React
- TypeScript
- REST API
- CSS

---

## What I Practised

This project helped me practise:

- fetching data from APIs in React
- working with TypeScript in React applications
- managing loading states
- implementing pagination
- handling user input and search queries
- structuring React components

## Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/MarinaHurricane/04-react-query.git
```

Navigate to the project folder:

```bash
cd REPO_NAME
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

Open the application in your browser:

```
http://localhost:3000
```
